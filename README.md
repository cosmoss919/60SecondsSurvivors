# 60초 생존자 (60 Seconds Survivors)
## 게임 기획안

---

## 📋 프로젝트 개요

### 기본 정보
- **프로젝트명**: 60초 생존자 (60 Seconds Survivors)
- **장르**: 타임어택 뱀파이어 서바이벌 류 액션
- **타겟 플랫폼**: Mobile (Android/iOS)
- **개발 기간**: 1주 (7일)
- **개발 인원**: 1인 개발
- **개발 엔진**: Unity 2022.3 LTS

### 프로젝트 목표
짧은 플레이타임(60초) 안에 최대한 많은 점수를 획득하는 타임어택 방식의 뱀파이어 서바이벌 류 게임으로, **모바일 최적화**와 **대량 오브젝트 처리**에 중점을 둔 포트폴리오 프로젝트

---

## 🎮 게임 컨셉

### 핵심 게임플레이
1. **60초 타임어택**: 제한된 시간 안에 최대한 많은 적을 처치
2. **자동 공격**: 플레이어는 이동만 조작, 공격은 자동 발사
3. **즉시 버프 시스템**: 적 처치 시 랜덤 아이템 드랍, 획득 즉시 능력치 상승
4. **운 + 실력**: 드랍되는 아이템에 따라 빌드가 달라지는 랜덤성
5. **최고 점수 갱신**: 리플레이를 통한 점수 경쟁

### 게임 플로우
```
타이틀 화면
    ↓
게임 시작 (60초 타이머 시작)
    ↓
플레이어 이동 + 자동 공격
    ↓
적 처치 → 아이템 드랍 (랜덤)
    ↓
아이템 획득 → 즉시 능력치 상승
    ↓
60초 종료
    ↓
결과 화면 (점수, 최고점수, 리플레이)
```

## 🎯 핵심 기능

### 1. 플레이어 시스템
- **이동**: 키맵핑, 조이스틱 기반 8방향 이동
- **자동 공격**: 가장 가까운 적을 향해 자동 발사
- **체력 시스템**: 적과 충돌 시 데미지, 체력 0이면 게임 오버

### 2. 적(Enemy) 시스템
- **AI**: 플레이어를 향해 지속적으로 추적
- **스폰 시스템**: 
  - 화면 밖 랜덤 위치에서 생성
  - 시간이 지날수록 스폰 빈도 증가 (난이도 곡선)
- **타입**: 기본 추적형 적 (시간 되면 2-3종류로 확장 가능)

### 3. 아이템/버프 시스템
적 처치 시 30-40% 확률로 아이템 드랍

**아이템 종류 (7가지)**:
- **공격 속도 증가**: 발사 간격 감소 (+15%)
- **데미지 증가**: 투사체 데미지 상승 (+20%)
- **이동 속도 증가**: 플레이어 이동 속도 상승 (+10%)
- **투사체 수 증가**: 한 번에 발사되는 투사체 +1개
- **관통력 증가**: 투사체가 적을 관통하는 횟수 +1
- **체력 회복**: 즉시 HP 30% 회복
- **무적 시간**: 3초간 무적 상태

### 4. 점수 시스템
- **기본 킬 점수**: 적종류에 따른 점수
- **최고 점수 저장**: PlayerPrefs로 로컬 저장

## 💻 기술 스택 및 구현 사항

### 핵심 기술 요소

#### 1. **Addressables System**
- **용도**: 에셋 번들 관리 및 메모리 최적화
- **적용 범위**:
  - 적 프리팹
  - 투사체 프리팹
  - 아이템 프리팹
  - 이펙트 프리팹
- **효과**: 빌드 사이즈 감소, 런타임 메모리 효율화

#### 2. **Object Pooling**
- **용도**: 빈번한 생성/삭제 최적화
- **적용 대상**:
  - Enemy Pool
  - Projectile Pool
  - Item Pool
- **효과**: GC Alloc 최소화, 프레임 드랍 방지

#### 3. **ScriptableObject**
- **용도**: 데이터 관리 구조화
- **데이터 종류**:
  - EnemyData (체력, 속도, 점수)
  - WeaponData (데미지, 공속, 투사체 수)
  - ItemData (버프 타입, 수치)
  - WaveData (스폰 빈도, 난이도 곡선)
- **효과**: 밸런싱 작업 용이, 유지보수성 향상

#### 4. **TextMeshPro**
- **용도**: UI 텍스트 렌더링
- **적용 범위**:
  - 점수 표시
  - 타이머
  - 콤보 카운터
  - 버튼 텍스트
- **효과**: 선명한 텍스트, 다양한 효과 적용 가능
